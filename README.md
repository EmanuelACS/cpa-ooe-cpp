# cpa-ooe-cpp
Semester 3 - Object Oriented Programming Using C++

## How to compile and run projects
### Single file
```
$ g++ fileName.cpp // compiles code to create a.exe
$ ./a.exe // run a.exe
```

### Multifile Project
```
$ g++ file1.cpp file2.cpp ... allCppFiles.cpp // include all .cpp files
$ ./a.exe
```
